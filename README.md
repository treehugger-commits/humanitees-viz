# humanitees-viz

Agent-published visualizations for Humanitees ops. Each subdirectory is one viz, served at:

`https://treehugger-commits.github.io/humanitees-viz/<slug>/`

**Do not hand-edit.** Files here are written by `~/humanitees/skills/publish-viz` via `~/humanitees/infra/scripts/publish-viz.sh`. The top-level `index.html` is regenerated on every publish.
